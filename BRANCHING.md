# Branching Strategy

- `main`: stable deploy branch
- `dev`: integration branch
- `feature/<scope>-<name>`: feature branch
- `fix/<scope>-<name>`: bug fix branch

Examples:
- `feature/sionna-smoke-runner`
- `feature/scene-converter-v1`
- `fix/material-assignment-bug`

Commit prefix:
- `feat: ...`
- `fix: ...`
- `chore: ...`
- `docs: ...`
